README.md

Foonathan Memory Vendor

Vendor package for foonathan/memory. This package will download, patch, build and install foonathan/memory for its use with Fast DDS.

Commercial support

Looking for commercial support? Write us to info@eprosima.com

Find more about us at eProsima’s webpage.

Quality Declaration

This package claims to be in the Quality Level 2 category, see the Quality Declaration for more details.

CHANGELOG

Changelog for package foonathan_memory_vendor

1.3.2 (2023-10-24)

  • Improve mechanism to find an installation of foonathan_memory (#67)

1.3.1 (2023-05-10)

  • Added support for QNX 7.1 build (#65)

1.3.0 (2023-02-14)

  • Update upstream to release 0.7-3 (#62)(#63)

1.2.2 (2022-12-09)

  • Fix CMake minimum required version (#60)

1.2.1 (2022-02-24)

  • Propagate [BUILD_SHARED_LIBS]{.title-ref} and [CMAKE_POSITION_INDEPENDENT_CODE]{.title-ref} to memory (#55)(#58)
  • Remove unused [foonathan_memory-config.cmake]{.title-ref} file (#57)

1.2.0 (2021-11-25)

  • Update upstream to release 0.7-1 (#49)

1.1.0 (2021-05-18)

  • QD Security Vulnerability Declaration: REP 2006 (#34)
  • Update vendor package Quality Declaration to QL3 (#35)
  • Add options instead of forcing them (#32)
    • Add build memory tools option
    • Add options for building tests and examples
    • Added option dependency validation
  • Fixed linter issues (#38)
  • Point to upstream quality declaration and bump to level 2 (#39)
  • Fixed test dependency (#41)
  • Use git apply instead of cherry-pick (#42)
  • Shorten some excessively long lines of CMake (#44)
  • Update package scripts for VS2019 (#45)
  • Drop support for VS2015 (#46)
  • Contributors: Chris Lalancette, Michel Hidalgo, Mauro Passerino, Miguel Company, Louise Poubel, Scott K Logan, Miguel Barro
