service_msgs/README.md

service_msgs

This package contains message types used by ROS services.

For more information about ROS services, check out the following links:

  • About ROS 2 interfaces
  • Understanding services

Messages (.msg)

  • ServiceEventInfo: Part of a service event message that contains information such as timestamp and event type. This is part of the Service Introspection feature of ROS 2 (see REP 2012).
