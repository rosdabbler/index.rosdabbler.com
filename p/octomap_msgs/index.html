octomap_msgs

ROS package to provide messages and serializations / conversion for the OctoMap library.

This repository continues from the groovy branch imported from SVN: http://alufr-ros-pkg.googlecode.com/svn/branches/octomap_stacks-groovy-devel/octomap_msgs/ See https://code.google.com/p/alufr-ros-pkg/ for the previous versions.
