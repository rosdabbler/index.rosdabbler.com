ros_ign_image/README.md

Image utilities for using ROS and Ignition Transport

This package provides a unidirectional bridge for images from Ignition to ROS. The bridge subscribes to Ignition image messages (ignition::msgs::Image) and republishes them to ROS using image_transport.

For compressed images, install compressed_image_transport and the bridge will publish /compressed images. The same goes for other image_transport plugins.
