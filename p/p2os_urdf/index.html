  • Use wheel descriptions from pioneer3dx_wheel.xacro. Remove invalid collision element from wheel definition. Comment out transmission element for now.
  • Remove visual names. Visual names prevented Gazebo from rendering material colors, as per this issue: https://bitbucket.org/osrf/sdformat/issues/132/parser-does-not-handle-urdf-material
  • Fix crash on model spawn in Gazebo. Remove zero-sized collision elements, as per this thread: http://answers.gazebosim.org/question/15816/gazebo-crashes-when-spawning-robot-from-urdf/
  • Contributors: Damjan Miklic
