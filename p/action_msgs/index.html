action_msgs/README.md

action_msgs

This package contains several messages and services useful for ROS 2 actions. More information about actions can be found on its design article.

For more information about ROS 2 interfaces, see docs.ros.org.

Messages (.msg)

  • GoalInfo: Goal identifier message, with a goal id and time stamp.
  • GoalStatus: Describes a goal’s current state machine status.
  • GoalStatusArray: An array of GoalStatus messages.

Services (.srv)

  • CancelGoal: Cancel Goals either by id and/or timestamp.

Quality Declaration

This package claims to be in the Quality Level 1 category, see the Quality Declaration for more details.

action_msgs

This package contains several messages and services useful for ROS 2 actions. More information about actions can be found on its design article.

For more information about ROS 2 interfaces, see docs.ros.org.

Messages (.msg)

  • GoalInfo: Goal identifier message, with a goal id and time stamp.
  • GoalStatus: Describes a goal’s current state machine status.
  • GoalStatusArray: An array of GoalStatus messages.

Services (.srv)

  • CancelGoal: Cancel Goals either by id and/or timestamp.

Quality Declaration

This package claims to be in the Quality Level 1 category, see the Quality Declaration for more details.
